CKEditor Samples » Mathematics

This sample shows usage of mathematical plugin. You can create or modify equations using TeX. TeX will be automatically replaced by widget when you put it <span class="math-tex">.

The output of the editor will be plain TeX with MathJax delimiters: \( and \), as in the code below:

<span class="math-tex">\( \sqrt{1} + (1)^2 = 2 \)</span>

To transform TeX into visual equation on target website the MathJax script must be included on it.
